Description:
This plugin was created to demonstrate how to create structures and enumerations using the Plugins SDK.  This code is provided as-is and there is no promise of it being fit for production.  However, it should be very explanatory and be a good learning example of how to create structures and enumerations.  If you find any bugs or have suggestions, please report them feedback system.

Topics this project covers:
How to make a plugin structure
 Structures within modules
 Structures as global code items
 Calling a method using the structure as a parameter

How to make a plugin enumeration
 Enums within modules
 Enums as global code items
