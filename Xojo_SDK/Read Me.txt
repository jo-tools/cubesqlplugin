About the Xojo Plugins SDK
===============================

This folder contains code and documentation to get you started making plug-in functions and controls for Xojo.  You'll find the following folders:

Documentation: this folder contains the SDK documentation in HTML format.

Examples: you'll find several subfolders here illustrating how to make Xojo plug-ins.

Glue Code: this folder contains C++ code you must add to your plug-in projects.

Includes: these files are included by the glue code (and most likely by your own plug-in code as well).
