tableName – table that column exists in columnName – the column to modify whichConstraint – "Primary Key", "Mandatory", etc... same text that displays in the properties listbox in the database schema editor oldConstraint – the old value (if boolean, this string will be either "True" or "False") newConstraint – the new value (if boolean, this string will be either "True" or "False")
