26 /**
27 * @brief Accesses the data of a string.
28 *
29 * This retrieves the contents of the string in a known encoding. If it is not
30 * possible to convert the string, it will return false.
31 *
32 * This function is more efficient than REALGetStringContents in the typical use
33 * case where a plugin needs the string's data in a known encoding.
34 *
35 * You are required to dispose of this string using REALDisposeStringContents when
36 * you are done with it.

38 * @note The behavior of this function changed in 2014r2. Previously it would
39 * return false given a NULL string.
40 * @param str The REALbasic string.
41 * @param encoding A REALbasic encoding identifier.
42 * @param outData The REALstringData structure to fill. Must not be NULL.
43 * @result Whether or not the data could be read.
44 * @version 2012r1
45 */
